# StickyPlane

## Status

Active (Personal Product)

## Category

AI Productivity / Knowledge Management

## Description

StickyPlane is an AI-powered note taking application focused on
capturing information instantly while making it searchable and useful
later. The product emphasizes fast note creation, AI-assisted
organization, semantic search, authentication, security, and
cross-platform usability.

## Primary Goal

Replace traditional note applications with an AI-first workflow where
users spend less time organizing notes.

## Major Features

-   AI generated titles
-   AI summaries
-   AI generated tags
-   Semantic search using embeddings
-   Fast indexing
-   Google authentication
-   GitHub authentication
-   Account linking
-   Password management
-   BetterAuth authentication
-   PWA support
-   Quick note creation
-   Secure account settings

## Tech Stack

Frontend - Next.js - React - TypeScript - Tailwind CSS

Backend - BetterAuth - Node.js

Database - PostgreSQL

Infrastructure - Embeddings - Vector search

# anyai

## Status

Published Open Source Library

## Category

Developer Tooling

## Description

anyai is a provider-agnostic AI runtime for TypeScript that exposes a
unified API across multiple LLM providers. Developers write code once
and swap providers without rewriting business logic.

## Supported Providers

-   OpenAI
-   Anthropic
-   Google Gemini
-   Mistral
-   Ollama
-   Additional adapters

## Goals

-   Unified API
-   Adapter architecture
-   Dynamic imports
-   Tree shaking
-   Strict TypeScript
-   ESM only
-   Capability-first design

## Tech Stack

-   TypeScript
-   Node.js
-   pnpm
-   Optional provider SDKs
